First Weeks at CodeFactory

I've been learning about web development at CodeFactory. Here's a summary of what I've learned so far and what I will learn in the upcoming weeks:

Git and GitHub

  • Understanding version control and the necessity of Git.
  • Installing and configuring Git
  • Making commits and following best practices and standards.
  • Working with GitHub repositories
  • Cloning projects from GitHub to the local machine.

HTML

  • Using HTML tags to create and structure websites.
  • Creating a simple HTML document and understanding its components.
  • Formatting text with headings and paragraphs.
  • Incorporating images and links into web pages.
  • Distinguishing between block-level and inline elements, using div and span.

CSS

  • Understanding the anatomy of Cascading Style Sheets (CSS).
  • Using internal, external, and inline styles.
  • Applying CSS selectors including element, ID, and class selectors.
  • Exploring the box model and its components: dimensions, margin, padding, and border.
  • Implementing different positioning methods: static, fixed, absolute, sticky, and relative.

Bootstrap

  • Integrating Bootstrap into web projects for responsive design.
  • Using Bootstrap's grid system for layout.
  • Applying Bootstrap components such as navbars, buttons, and cards.
  • Customizing Bootstrap styles with CSS.
  • Utilizing Bootstrap utilities for spacing, alignment, and visibility.

JavaScript

  • Understanding the basics of JavaScript and its role in web development.
  • Using variables, functions, and control structures.
  • Manipulating the DOM to create interactive web pages.
  • Handling events and user interactions.
  • Working with arrays and objects for data management.

TypeScript

  • Understanding the advantages of using TypeScript over JavaScript.
  • Setting up a TypeScript environment and compiling TypeScript code.
  • Using type annotations and interfaces for type safety.
  • Applying TypeScript features like classes, modules, and generics.
  • Integrating TypeScript with JavaScript libraries and frameworks.

Angular

  • Setting up an Angular project and understanding its structure.
  • Creating and using components for modular development.
  • Implementing data binding and directives for dynamic templates.
  • Working with services and dependency injection for scalable applications.
  • Managing application state with Angular's reactive forms and RxJS.
